Raw data were converted to ISMRMRD format using siemens_to_ismrmrd (https://github.com/ismrmrd/siemens_to_ismrmrd)

Raw data file raw_IRSE.mrd is hosted here: https://owncloud.fraunhofer.de/index.php/s/6lrZuFPqFxeBkrv

Reconstructed image file: recon_IRSE.h5
